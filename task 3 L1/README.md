# Calculator Project

A modern, responsive calculator built with HTML, CSS, and JavaScript. This project demonstrates the implementation of a fully functional calculator with a clean user interface and smooth interactions.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Decimal point support
- Clear functionality
- Operation history display
- Error handling for division by zero
- Responsive design
- Modern UI with CSS Grid

## Technologies Used

- HTML5
- CSS3 (with CSS Grid)
- JavaScript (ES6+)

## Usage

The calculator supports the following operations:

- **Numbers (0-9)**: Input numbers
- **Operators**:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- **Special Functions**:
  - Clear (C): Resets the calculator
  - Decimal Point (.): Adds decimal numbers
  - Equals (=): Shows the result

## Features in Detail

1. **Display System**:
   - Main display shows current number/result
   - Secondary display shows the current operation

2. **Operation Handling**:
   - Supports chaining operations (e.g., 2 + 3 + 4)
   - Maintains operation history
   - Handles decimal numbers
   - Error handling for invalid operations

3. **User Interface**:
   - Clean, modern design
   - Responsive layout
   - Color-coded buttons
   - Smooth transitions and hover effects

## Example Calculations

3. Chaining Operations:
