# Personal Portfolio Website

A responsive personal portfolio website built with HTML and CSS, showcasing professional experience and skills.

## Features

- Responsive design for all devices
- Modern UI with smooth transitions
- Sections for About, Skills, Projects, and Contact
- Interactive elements with hover effects
- Clean and professional layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Poppins)

## Project Structure

```
task 1 L1/
├── index.html      # Main HTML file
├── style.css       # CSS styles
├── pp.jpg          # Profile picture
└── README.md       # Project documentation
```

## Setup

1. Clone the repository:
```bash
git clone https://github.com/sirishax/codsoft.git
```

2. Navigate to the project directory:
```bash
cd codsoft/task 1 L1
```

3. Open `index.html` in your web browser

## Customization

1. Open `index.html` and update:
   - Personal information
   - About section
   - Skills
   - Projects
   - Contact details

2. Modify `style.css` to:
   - Change color scheme
   - Adjust animations
   - Customize layout

## Browser Support

- Chrome (latest)
- Firefox (latest)

- Edge (latest)
